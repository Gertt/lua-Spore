Overview

lua-Spore is an implementation of SPORE (Specification to a POrtable Rest Environment).

References

Some specifications of service are available at http://github.com/franckcuny/spore.

The Perl5 implementation is available at http://github.com/franckcuny/net-http-spore.

Status

lua-Spore is in early stage.

It's developed for Lua 5.1.

Download

lua-CodeGen source can be downloaded from GitHub or Lua Forge.

Installation

lua-CodeGen is available via LuaRocks:

luarocks install lua-codegen

or manually, with:

make install

Test

The test suite requires the module lua-TestMore.

make test
